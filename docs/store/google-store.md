### ko (대한민국 한국어)

**제목** (최대 30자)
`스핀 휠 - 랜덤 선택기`

**간략한 설명** (최대 80자)
`화려한 스핀 휠로 무엇이든 결정하세요. 커스텀 항목, 저장, 히스토리 지원.`

**설명** (최대 4000자)

오늘 점심 뭐 먹을지, 게임에서 누가 먼저 할지, 여행지는 어디로 갈지. 결정이 어려울 때 스핀 휠이 대신 골라드립니다!

스핀 휠은 결정을 즐거운 게임으로 만들어주는 랜덤 선택기입니다.

■ 완전 커스텀 가능
- 내 상황에 맞는 항목 직접 추가 (최대 20개)
- 항목 편집, 삭제 자유롭게
- 여러 개의 휠 프리셋 저장 (식당 메뉴, 게임 순서, 할 일 등)
- 휠마다 이름 설정 가능

■ 화려한 스핀 경험
- 물리 기반 감속 애니메이션 (관성으로 서서히 멈춤, 4500ms easeOut)
- 컬러풀한 10가지 색상 자동 배분
- 버튼을 누르면 glow 효과와 함께 스핀 시작
- 결과가 탄성 있게 팝업으로 표시

■ 여러 휠 관리 & 결과 히스토리
- 식당 선택용, 게임 순서용, 팀 배정용 등 별도 저장
- 홈에서 색상 미리보기로 한눈에 구분
- 각 휠마다 최근 결과 최대 20개 자동 기록
- 가로 스크롤로 역대 당첨 항목 확인

■ 데이터 & 오프라인
- 인터넷 없이 완전 오프라인 작동
- 모든 데이터는 기기 내부에 안전하게 저장

■ 광고 안내
- 배너 광고 포함 (완전 무료)
