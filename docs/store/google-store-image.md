## 2. 캡처 체크포인트

- 촬영 전 준비: 식당 메뉴 휠 (Pizza/Sushi/Burger/Tacos/Ramen), 게임 순서 휠 (Player 1~4), 여행지 휠 (Jeju/Tokyo/Bali/Paris/NY) 미리 생성
- 홈 화면: 최소 3개 휠이 리스트에 표시된 상태, 각 카드에 마지막 결과 표시
- 스핀 중 스크린샷: 타이밍이 어려우면 스크린 녹화 후 프레임 추출
- 결과 카드: 당첨 항목이 "Pizza" 또는 "Tokyo"처럼 읽기 쉬운 영문 항목으로 나왔을 때 캡처
- 편집 시트 스크린샷: 항목이 3개 이상 보이는 상태
- 배너 광고 영역이 하단에 표시된 상태 (정책 준수)
- 기기: Pixel 7 시리즈 또는 Samsung Galaxy S 시리즈 권장

## 3. 이미지 사양

- 권장 해상도: `1080 x 1920` (9:16)
- 형식: PNG 또는 JPG
- 권장 장수: 8장
- 파일 용량: 이미지당 8MB 이하

## 4. 제출 전 체크리스트

- [ ] 스크린샷 8장 모두 1080x1920 px 확인
- [ ] 파일 크기 각 8MB 이하 확인
- [ ] 상태바 클린 상태 (알림 없음, 배터리 100%, 시간 정각)
- [ ] 스핀 중 스크린샷: 휠이 실제로 회전하는 프레임 사용
- [ ] 결과 카드: 당첨 항목 텍스트가 명확히 읽히는지 확인
- [ ] 배너 광고 영역이 하단에 표시된 상태 (정책 준수)
- [ ] 편집 시트 스크린샷: 항목이 3개 이상 보이는 상태
